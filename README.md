# Auto fuel
система автаматического полива

### необходимые детали:
- **кнопки** - 3 шт
- **ESP-12E** - по идее полжно работать и на других контролерах схожих с ESP-8266 но разработка велась именно с 12E
- **жидко кресталический дисплей C162-198B с I2C контролером** - желательно 2 строки по 16 синволов
- **реде на 220В с упраляющим током от 3В** - при разработке использовал реле SSR-25 DA
- **водяной насос** - 220В не более 5500 ват    

### пины подключения:
- `D1` - выход для запитки реле при вкючении подаеться +3v (зависит от контролера)
- `D2` - кнопка включения/выключения полива не зависимо от таймера интервала
- `D4` - кнока увеличивающая время на 1
- `D5` - конпка уменьшающая время на 1
- `D6` и `D7` - подключение ЖК дисплея (расчитано на I2C дислей 16x2)

### API
- `<IP>:9000/` - основной WEB инерфейс для упрвления
- `<IP>:9000/toggle` - включение/выключение
- `<IP>:9000/getstatus` - получение статуса, включен или нет
- `<IP>:9000/handle_time_nums` - ожидает POST запроса с целым числом, задает таймер 
- `<IP>:9000/setup_time_interval` - ожидает POST запроса с целым числом, задает интервал между автоматическими включениями
